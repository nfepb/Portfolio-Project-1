<h1>Masterminding Board Games</h1>

The Masterminding Board Games website is a landing pager where gamers in Dublin can discover certain types of games and to meet people that share the same hobby.

Through this webiste, users will be able to discover 4 types of board games, some pictures illustrating some of people's most favourite board games, and a signup page where people can register to participate in gaming sessions.

  <h2>Testing</h2>
            
   <li>The page was tested on 3 different browsers: Chrome, Mozilla, and Safari</li>
              <li> The components are responsive on tablet and smartphone. Unfortunately, I did not manage to re-arrange all of them in the given time. </li>
                <li> Users can easily navigate between the different pages through the menu that can be found at the top of every page</li>
                  <li> All the content can be easily assimilated and is readable for colourblind people too. </li>
                    <li> The form also responds to the set criteria. For example, the email field will only accept an input in the form of "xxx@xxx.com".</li>
